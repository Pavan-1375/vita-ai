YouOkay? Project

Full Architecture & Design

Health PWA: Symptom AI, Trackers, Voice Consult

React + Vite + FastAPI + Gemini AI

Overview

  • Frontend PWA (CWD): React 19 + Vite + TS + Tailwind + PWA
  • Backend Server (Desktop/hackathon/): Python FastAPI + ML + Gemini
  • Connection: Vite proxy /api → backend:5000
  • Run: npm run dev:all (auto-stack)

Ports: FE :3000 | BE :5000

File Structure

Frontend (vita-ai/)
├── public/     # PWA: manifest, icons, SW
├── src/
│   ├── components/ # HomeScreen, ConsultScreen (AI)
│   ├── lib/api.ts  # /api/* (proxied)
│   └── App.tsx     # SPA screens
├── scripts/
│   └── start-stack-v3.ps1 # Stack orchestrator
└── vite.config.ts  # Proxy config

Backend (Desktop/hackathon/)
├── api.py         # FastAPI: /predict, /claude
└── ML model

Frontend Deep Dive

  • SPA: Screens (home/consult/remedies/insights) via state + AnimatePresence
  • HomeScreen: Hydration/breathing trackers (localStorage), tips carousel
  • ConsultScreen (Core):
    • Symptoms text/voice → POST /api/predict
    • Response: Disease, Confidence, Triage, Precautions[], Remedies[]
    • Chat: /api/claude (Gemini)
    • Voice: Web Speech API
  • PWA: Offline trackers/fallbacks

Backend Deep Dive

  • FastAPI (uvicorn api:app --port 5000)
  • /predict POST {symptoms:[]}: ML model → structured JSON
  • /claude POST: Gemini AI reasoning (VITE_GEMINI_API_KEY)
  • Auto-reload: backend_dev_runner.py (watches files)

Connection Flow

Frontend fetch('/api/predict') → Vite Proxy → Backend /predict

vite.config.ts:
proxy: {
  '/api': {
    target: 'http://127.0.0.1:5000',
    rewrite: /^\/api/ → ''
  }
}

User Flow

  1. npm run dev:all → Backend (:5000) + Vite (:3000) + Open PWA
  2. Home: Track hydration/breathing, "Analyze Symptoms"
  3. Consult:
    • Voice/text symptoms
    • → /api/predict → ML JSON
    • Display: Disease/Conf/Triage/Precautions/Remedies
    • Chat fallback: /api/claude
  4. Offline: Local hardcoded predictions
